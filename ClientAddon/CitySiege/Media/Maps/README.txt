!! CRITICAL: WoW 3.3.5 ONLY LOADS BLP FORMAT !!

TGA/PNG files will NOT work. You MUST convert to BLP.

=== CONVERSION TOOL (EASY) ===
Download BLPConverter:
https://www.wowinterface.com/downloads/info8127

Steps:
1. Download and extract BLPConverter.exe
2. Run BLPConverter.exe
3. Drag your TGA files into the window
4. Click "Convert to BLP"
5. Save the .blp files here in Maps folder

=== REQUIRED BLP FILES ===
- Stormwind.blp
- Ironforge.blp
- Darnassis.blp (note spelling!)
- TheExodar.blp
- Ogrimmar.blp (note spelling!)
- Undercity.blp
- ThunderBluff.blp
- SilvermoonCity.blp

Recommended size: 1024x768 or 512x512 pixels
Format: BLP0 (uncompressed) or BLP1 (DXT compression)
